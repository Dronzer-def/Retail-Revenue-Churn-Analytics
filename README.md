# Retail Revenue Analytics & Customer Churn Prediction

## Project Overview

This project analyzes retail transaction data to understand customer purchasing behavior, identify valuable customer segments, and predict customer churn using machine learning techniques.

The project combines data cleaning, exploratory data analysis (EDA), SQL analysis, RFM customer segmentation, K-Means clustering, and churn prediction to generate actionable business insights.


## Dataset

Dataset: Online Retail Dataset

The dataset contains:

- Customer transactions
- Product information
- Invoice details
- Purchase quantities
- Revenue information
- Transaction dates

## Exploratory Data Analysis

Performed analysis on:

- Daily Sales Trends
- Top Countries by Sales
- Top Customers by Spending
- Revenue Patterns

Key insights were extracted to understand customer purchasing behavior and business performance.


## RFM Analysis

RFM stands for:

- Recency
- Frequency
- Monetary

The analysis was used to evaluate customer value and purchasing behavior.


## Customer Segmentation

K-Means clustering was applied on RFM metrics.

Customer segments identified:

- Champions
- Loyal Customers
- Potential Loyalists
- Lost Customers

These segments help businesses create targeted marketing strategies.


## Churn Prediction

A churn prediction model was developed to identify customers likely to stop purchasing.

Models used:

- Logistic Regression
- Random Forest

Evaluation metrics:

- Accuracy
- Precision
- Recall
- F1 Score


## Key Business Insights

- High-value customers were identified through RFM analysis.
- Customer segments were created using K-Means clustering.
- Churn prediction models helped identify at-risk customers.
- Business recommendations were developed for customer retention and revenue growth.
